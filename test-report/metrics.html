CCN - Cyclomatic Complexity

MI Formula

CCN = Number of decision points + 1

Decision Points

if, elseifConditional branches
for, foreach, while, doLoop constructs
caseSwitch case (with condition)
catchException handlers
&&, ||Boolean operators
?:Ternary operator
??Null coalesce operator

A 1-4 : Simple, low risk, easy to test
B 5-7 : Moderate complexity
C 8-10 : Complex, moderate risk
D 11-15 : High complexity, hard to test
F 16+ : Very high risk, untestable
