How to Generate Coverage

Use PHPUnit with the --coverage-clover option to generate a coverage report in Clover XML format:

# PHPUnit (Clover format)
./vendor/bin/phpunit --coverage-clover coverage.xml

# Then run PhpQuality with coverage
docker run --rm \
  -v $(pwd):/project \
  amoifr13/phpquality \
  analyze --source=/project/src --coverage=/project/coverage.xml
